# 📊 Personal Expense Tracker

A simple command-line application built using Python to track and analyze daily expenses.

---

## 🚀 Features

* Add new expense
* Delete wrong expense entry
* View monthly expense summary
* View category-wise expenses
* View overall yearly expenses
* Display yearly category-wise bar chart

---

## 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* CSV file handling

---

## 📂 Project Structure

expense-tracker/

* expense_tracker.py → Main program
* expenses.csv → Stores expense data
* README.md → Project details

---

## ▶️ How to Run

1. Install required libraries:

   ```bash
   pip install pandas numpy matplotlib
   ```

2. Run the program:

   ```bash
   python expense_tracker.py
   ```

---

## 📌 Menu Options

1. Add Expense
2. Delete Expense
3. Monthly Summary
4. Category-wise Summary
5. Yearly Summary
6. Category-wise Bar Chart
7. Exit

---

## 📊 What This Project Does

* Stores expenses in a CSV file

* Shows summaries and simple insights
